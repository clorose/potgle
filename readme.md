## Usage

```javascript
const PORTS = {
    React: "3000",
    Vite: "5173",
    FastAPI: "8000",
    "Node.js": "4000",
    "Spring Boot": "8080",
    // Add your ports:
    // ServiceName: "port"
    // or
    // "Service Name": "port"    // 띄어쓰기 있으면 따옴표 필수
};
```
