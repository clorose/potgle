# Portgle
I'm just too lazy to type ports.

## Usage
1. Fork this repository
2. Edit `ports.js` to add your commonly used ports:
```javascript
const PORTS = {
    React: "3000",
    Vite: "5173",
    FastAPI: "8000",
    "Node.js": "4000",
    "Spring Boot": "8080",
    // Add your ports:
    // ServiceName: "port"
    // or
    // "Service Name": "port"    // Quotes required if name contains spaces
};
```
